TECHNOLOGY PARTNER

Analytics Creator

analyticscreator.com ↗

What It Is

Analytics Creator (AC) is an enterprise data warehouse automation platform built for SQL Server and SSIS environments. It automates the generation of Kimball-style dimensional models and Data Vault structures — including staging, historisation, and persistence layers — from a configuration-driven interface rather than hand-coded SQL.

How ProcessBI Uses It

For complex data estate engagements — particularly where the client operates SQL Server or is building a structured warehouse alongside their Microsoft Fabric layer — ProcessBI leverages Analytics Creator to enforce structural consistency, reduce warehouse build time, and ensure every data model is auditable and change-managed from the start.

Staging Layer

Import and staging table generation (STG/IMP schemas) from source system definitions — no hand-writing CREATE TABLE scripts.

Historisation

SCD2 historisation (HIST schema) configured through a GUI — change detection, effective dating, and validity windows handled automatically.

Deployment

DACPAC and ADF pipeline generation with full deployment to SQL Server or Azure environments — integrated into the CI/CD pipeline.

ProcessBI's process-first methodology pairs naturally with Analytics Creator's model-first data warehouse approach. Where we define and agree the data requirements through process analysis, Analytics Creator ensures those requirements translate into a robust, maintainable warehouse architecture — not a tangle of bespoke stored procedures.
